Hello from the Outside is an Android app my team built for Outside Hacks IV that can be used to locate friends at crowded events. We used Google Maps and Firebase to store locations of friends, as well as vibrations to sense how close your friends are. I focused on storing the various users' names using Shared Preferences.

This program was my team's winning submission for HKN BearHacks 2016. The program is built with Python and stems from a workshop hosted by HKN demoing a sample computer vision app. It has the ability to swap faces, as well as overlay a face with a different image, using the OpenCV library and image resizing to do so.

Project Scale

Project Scale is my team's submission for HackDavis 2017. It is the combination of a website based in PHP and an Android app. The project is designed to help developers find others to work with them on projects. I focused on Google sign-in and experimenting with nav drawers for the app, as I had only a few hours to build it.
