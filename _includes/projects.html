TileWarp

TileWarp is a puzzle app that can be used as both a game and a learning tool. I developed it in eleventh grade with the help of two other students, a marketer and a designer, as a way to implement my Java knowledge. To build it, I used Android Studio, focusing on switch statements and array lists. It is now available in the Google Play Store.
